### Task 2: Widget docs generation command

**Files:**
- Create: `cmd/mxcli/cmd_generate_widget_docs.go`

**Step 1: Add `mxcli generate widget-docs` command**

Cobra command that:
1. Scans `widgets/*.mpk` in project dir → `mpk.ParseMPK()` for each
2. For each widget, generates markdown with:
   - Widget ID, name, version
   - Property table (from PropertyDef: key, type, required, default, category, description)
   - MDL example: `CUSTOMWIDGET name1 (WidgetType: 'widget.id')`
   - `<!-- AUTO-GENERATED -->` markers
3. Writes to `.claude/skills/widgets/<name>.md` (or `.ai-context/skills/widgets/`)
4. Generates `_index.md` with summary table

**Step 2: Wire into `mxcli init`**

In `cmd/mxcli/init.go`, call widget docs generation after skill sync.

**Step 3: Build and test**

```bash
bin/mxcli.exe generate widget-docs -p D:/gh/poc-carrefour/ORION-AI-Production-Order.mpr
ls D:/gh/poc-carrefour/.ai-context/skills/widgets/
```
